## Short Description

Open Local Digital Twin is an Apache-2.0, self-hosted runtime for building
public-data city baselines, preserving source evidence, separating public,
provider, inferred, and authority-grade layers, and exposing interoperable city
intelligence through open standards.

## SDG Relevance

The solution supports SDG 9, SDG 11, SDG 13, SDG 16, and SDG 17 by helping
cities and public-interest teams build interoperable civic-data infrastructure,
improve evidence-based urban planning, support climate and resilience analysis,
and strengthen transparent institutional decision-making.

## Platform Independence

The runtime is self-hosted and based on Node.js, Express, Next.js,
PostgreSQL/PostGIS, Docker Compose, MapLibre, and open geospatial/civic-data
standards. Closed provider data, hosted services, proprietary extensions, and
private deployments are optional and are not required for the public-data
baseline.

## Data Import And Export

The solution imports and exports non-PII city data through non-proprietary
formats and APIs, including GeoJSON, CSV-compatible workflows, JSON, JSON-LD,
Mapbox Vector Tiles, DCAT, OGC API Features, NGSI-LD entity projections, and
FIWARE-compatible context workflows.

## Privacy Evidence

Privacy documentation:

https://github.com/polisplexity/open-local-digital-twin/blob/main/PRIVACY.md

The public baseline does not require personal microdata. Deployments adding
private or personal data must configure lawful access, retention, access
control, publication review, and jurisdiction-specific compliance.

## Standards And Best Practices

The runtime aligns with DCAT, OGC API Features, NGSI-LD, FIWARE-compatible
context workflows, PostGIS-backed geospatial storage, source provenance,
license/attribution metadata, bounded payloads, and vector tiles for city-scale
rendering.

## Do No Harm Evidence

Do-no-harm documentation:

https://github.com/polisplexity/open-local-digital-twin/blob/main/DO_NO_HARM.md

The project separates public evidence, inferred outputs, provider layers, and
authority-grade decisions. It avoids treating open data or AI inference as
official truth without human and institutional review.
